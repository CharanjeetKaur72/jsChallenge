30 Days Java Script Challenge

at https://www.chaicode.com

Right click anywhere on the page, select the menu option - inspect. Then view the result in console

  1. Declare a variable using var, assign a number, and log the value to the console.
  2. Declare a variable using let, assign it a string, and log the value to the console.
  3. Declare a variable using const, assign it a boolean value, and log the value to the console.
  4. Create variables of different data types (number, string, boolean, object, array) and log each variables type using the typeof operator.
  5. Declare a variable using let, assign it an initial value, reassign a new value, and log both values to the console.
  6. Try reassigning a variable declared with const and obsereve the error.
  1. Perform basic arithmetic operation (addition, subtraction, multiplication, division, remainder) on two numbers and log the result.
  2. Perform assignment operation += to add a number to a variable and , -= to subtract a number from a variable. Log the result.
  3. Compare two numbers using different comparison operators and combines the conditions using logical operators, logging the result.
  4. Use ternary operator to determine if a number is positive or negative and log the result.
  1. Use if-else statements to check if a number is positive, negative or zero.
  2. Use if-else statements to check if a person is eligible to vote (age >=18).
  3. Use nested if-else statements to find largest of three numbers.
  4. Use switch-case statement to determine day of the week, based on a number(1-7).
  5. Use switch-case statement to assign a grade ('A", 'B', 'C', 'D', 'F') based on a score.
  6. Use ternary operator to check, if a number is odd or even.
  7. Using multiple conditions check, if a year is leap year.
